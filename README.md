- Start/Restart and Pause button.
- Three ways to play 
  - Arrow keys 
  - Drag and drop the paddel 
  - mouse motion 
- Hitting the stone twice to break it.

- Lucky stones 
  - randomly appearing, causing an immediate win/lose .
- Animation background

- Sound effects
